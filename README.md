# main-web-resource

## 项目简介

这是一个用于归档和发布多个独立 HTML 页面的仓库。页面内容涵盖不同类型的静态网页资源，适用于学习、参考或直接使用。本仓库不针对任何特定域名或网站，仅作为 HTML 页面的集中存放与版本管理。

## 目录说明

```
main-web-resource/
├── pages/          # 存放所有独立 HTML 页面
├── assets/         # 页面依赖的静态资源（图片、样式、脚本等）
├── docs/           # 相关文档或说明文件（可选）
└── README.md       # 本文件
```

- `pages/`：每个 HTML 文件独立存在，命名清晰，便于查找。
- `assets/`：按需整理，与对应页面保持关联。
- `docs/`：存放额外的使用说明或设计文档。

## 页面归档说明

- 所有页面均为静态 HTML，不依赖后端服务。
- 页面之间相互独立，无交叉引用。
- 归档时尽可能保留原始结构与样式，必要时进行兼容性调整。
- 每个页面在提交时附带简要说明（可在 commit 信息中体现）。

## 维护说明

- 欢迎提交 Issue 报告问题或提出改进建议。
- 如需新增页面，请将 HTML 文件放入 `pages/` 目录，并确保资源路径正确。
- 请保持目录结构整洁，避免冗余文件。
- 本仓库不做频繁更新，以稳定归档为主。

## 使用方式

1. 克隆或下载本仓库。
2. 在 `pages/` 目录下找到所需 HTML 文件。
3. 直接在浏览器中打开即可查看效果。
